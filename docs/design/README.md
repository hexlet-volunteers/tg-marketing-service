# Дизайн tgpulse — макеты и спецификация

Здесь лежит дизайн-референс продукта **tgpulse** (аналитика Telegram-каналов с AI). Это источник истины для фронтенд-задач: по нему собираются все экраны.

## Файлы

| Файл | Что это |
|------|---------|
| [`tgpulse-prototype.html`](tgpulse-prototype.html) | **Интерактивный hi-fi прототип** — все 14 экранов и 10 модалок с навигацией и логикой. Открыть в браузере. |
| [`handoff-spec.md`](handoff-spec.md) | Полная спека: экраны, поведение, состояние, дизайн-токены (цвета/типографика/радиусы) и **маппинг на компоненты Mantine**. |
| `support.js` | Рантайм дизайн-прототипа (нужен для `tgpulse-prototype.html`). Не трогать. |

## Как открыть прототип

```sh
open docs/design/tgpulse-prototype.html      # macOS
# или просто перетащить файл в браузер
```

Навигация между экранами — по сайдбару и кнопкам внутри прототипа. Шрифты (Manrope/JetBrains Mono) подтягиваются с Google Fonts; без интернета прототип откроется на системных шрифтах — на вёрстку это не влияет.

## Важно для фронтенда

- Экраны нужно **пересоздать на реальной библиотеке Mantine** (`@mantine/core`), а не копировать инлайн-стили из прототипа.
- Соответствие «блок прототипа → компонент Mantine» и все дизайн-токены — в [`handoff-spec.md`](handoff-spec.md).
- Вёрстка — mobile first.
